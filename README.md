- Google Apps Script which copies [kanazawa.rb events](https://meetup.kzrb.org/) to spreadsheet.
- Using [clasp](https://github.com/google/clasp)
- Using TypeScript

## Requirements

- `npm install -g @google/clasp`

## Usage

- `clasp login`
- prepare `.clasp.json`
  - `{"scriptId":"XXXXXXXXXXXXXXXXXXXXXX"}`
- `clasp push`
